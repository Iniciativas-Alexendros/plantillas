# Configuración Claude Code · [Proyecto]

Este directorio `.claude/` configura Claude Code para trabajar en este proyecto.

## Estructura

```
.claude/
├── CLAUDE.md       ← Reglas y contexto del proyecto
├── settings.json   ← Permisos y configuración
└── mcp.json        ← MCP servers del proyecto
```

## Uso

Copia esta plantilla a tu proyecto:

```bash
cp -r proyecto ./.claude
```

Luego edita `CLAUDE.md` con las reglas específicas de tu stack.
