# Custom Tools

Directorio para herramientas personalizadas del agente.

Coloca aquí scripts o wrappers que extiendan las capacidades base del agente
más allá de las tools estándar de Claude Code.

## Convenciones

- Un archivo por tool: `nombre-tool.sh` o `nombre-tool.py`
- Incluir shebang y descripción en las primeras líneas
- Documentar inputs/outputs esperados
